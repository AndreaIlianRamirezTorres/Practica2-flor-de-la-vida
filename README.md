# Practica2-flor-de-la-vida
En este repositorio se explica el procedimiento para realizar una "flor de la vida" utilizando la herramienta blender
Para hacer esta practica se recurrira al uso del lenguaje phyton atravez de blender,
para empezar se requiere tener instalado blender, en caso de no contar con blender este se debe de instalar desde el sitio oficial,
una vez que el programa esta instalado se abre mostrando el entorno principal
